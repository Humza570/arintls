# Arintls — Corporate Website

Official website for **Arintls** — built with **Next.js** and deployed on **Hostinger**.

🌐 **Live:** [arintls.com](https://arintls.com/)

## Features

- ⚡ Fast, SEO-optimized pages with Next.js App Router
- 📱 Fully responsive — mobile, tablet, desktop
- 🎨 Modern UI with Tailwind CSS
- 🚀 Auto-deployed via Hostinger

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js |
| Styling | Tailwind CSS |
| Deployment | Hostinger |
| Language | TypeScript / JavaScript |

## Live Demo

[https://arintls.com](https://arintls.com/)

## Installation

```bash
git clone https://github.com/Humza570/arintls.git
cd arintls
npm install
npm run dev
```
